# theoriginalweb
theoriginalweb is a simple hosting-ish service made to mimic the old geocities. it isnt actually real hosting though. it uses redirects and subdomains on IONOS. you have to provide hosting yourself (e.g. via neocities.org, github.io, docs.google.com, or editpad.org.) and you can contact elitaylor120@gmail.com for a FREE subdomain from my website.
